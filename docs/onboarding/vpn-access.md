# VPN Access Guide
To access internal resources and production environments, you must connect to the corporate VPN.

* **Software:** Download and install the [Company VPN Client] from the internal software portal.
* **Authentication:** Use your corporate SSO credentials. Multi-Factor Authentication (MFA) is required for every connection.
* **Connection Steps:**
    1. Launch the client.
    2. Select the appropriate gateway (e.g., `prod-vpn` or `dev-vpn`).
    3. Enter your credentials and approve the MFA push notification.
* **Troubleshooting:**
    * Ensure your internet connection is stable.
    * If you receive an "Authentication Failed" error, verify your password or reset it via the IT self-service portal.
    * For persistent issues, contact the IT Helpdesk via the #it-support Slack channel.
