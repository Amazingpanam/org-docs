# New Staff Onboarding
Welcome to the team! Please complete the following steps during your first week:

* **Account Setup:** Ensure you have access to your company email, Slack, and Jira/Confluence.
* **Development Environment:** Clone the core repositories and follow the `README.md` in each project to set up your local environment.
* **Security Training:** Complete the mandatory security awareness modules provided via the HR portal.
* **Access Requests:** If you require specific cloud access (AWS/Azure), submit a request ticket to the DevOps team.
* **Documentation:** Familiarize yourself with our internal Wiki for coding standards and deployment workflows.
