# Terraform
An Infrastructure as Code (IaC) tool used to define, provision, and manage cloud infrastructure.

* **Declarative:** You define the "desired state" of your infrastructure, and Terraform handles the creation/updates.
* **State Management:** Uses a `terraform.tfstate` file to map real-world resources to your configuration.
* **Providers:** Plugins that interact with cloud APIs (AWS, Azure, GCP, etc.).
* **Key Commands:** `terraform init`, `plan`, `apply`, `destroy`.
