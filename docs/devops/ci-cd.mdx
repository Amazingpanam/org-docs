# CI/CD Overview
Continuous Integration (CI) and Continuous Deployment (CD) automate the software delivery process.

* **CI (Continuous Integration):** Developers frequently merge code into a shared repository. Automated builds and tests run to detect errors early.
* **CD (Continuous Deployment/Delivery):** Automated pipelines release code changes to production or staging environments, ensuring the software is always in a deployable state.
* **Key Tools:** Jenkins, GitHub Actions, GitLab CI, CircleCI.
