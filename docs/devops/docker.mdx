# Docker
A platform for developing, shipping, and running applications in containers.

* **Containers:** Lightweight, standalone, executable packages that include everything needed to run software (code, runtime, libraries).
* **Dockerfile:** A script containing instructions to build a Docker image.
* **Images vs. Containers:** An image is the read-only template; a container is the running instance of that image.
* **Key Commands:** `docker build`, `docker run`, `docker ps`, `docker stop`.
